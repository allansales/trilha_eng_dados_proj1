# trilha_eng_dados_proj1
Projeto 1 da formação da trilha de engenharia de dados da Compass UOL.

# Setup
Clona o projeto

Criar venv

instalar requirements

fazer uma cópia de .env.example 
renomear a cópia para .env
colocar as chaves da azure no .env

# Execução
## Pipeline
executar na raiz do projeto: python -m src.orchestration.pipeline

## Análise
executar na raiz do projeto: python -m analytics.eda
